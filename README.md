---

## 🚀 Getting Started

1. **Clone this repository:**
   ```bash
   git clone https://github.com/charizkyle/my_portfolio.git
    ```

2. **Open in VS Code**
3. **Run your project**

   * If static HTML/CSS → just open `index.html` in your browser
   * If Flask/Django → run `python app.py` or `python manage.py runserver`
